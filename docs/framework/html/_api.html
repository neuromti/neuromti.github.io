Main API¶

The experiment api is based on several submodules which wrap themselves the C++ Wrappers. The api is mainly use for creating experimental tasks. A library of several differnt tasks has already been created in the Tasks sumodules.

Example¶

The following example would create a task that presents a cross for five seconds on monitor 2:

import experiment.api as xp
Cue = xp.Cue(xp.SOUND['peep'], xp.STATIC['cross'], 5)
xp.Screen.open()
Cue.present()
xp.Screen.close()

Submodules¶

experiment.protocol This module contains classes to control the low of an experiment
experiment.device This module contains classes resembling devices
experiment.stimulus This module contains classes resembling acoustic and visual stimuli
experiment.classifier This module contains classes resembling online classifiers
