experiment.tasks.emg.mvc¶

Measure maximum voluntary contraction

Documentation¶

experiment.tasks.emg.mvc.start(chanOfInterest='EDC_L', numberOfRuns=1, saveDir='C:\\projects\\recording', rwksp='C:\\projects\\config\\bvr_workspace.json')[source]¶

Notes

A visual emg-contraction task for a single muscle to measure maximum voluntary contraction. It presents a vertical red bar representing the current contraction power. Whenever this bar reaches a horizontal target line at 100%, the bar turns green, and maximum value is updated. In that regard, the target is adaptive. It resets after each trial.

Parameters:
  • chanOfInterest (the channel used for feedback) –
  • numberOfTrials (A trial consists of 10s feedback and 4s rest) –
  • saveDir (where the runFile will be stored) –
  • rwskp (a json file containing the EEG workspace) –

side effect

  • saves a Brainvisionrecorder eeg file to saveDir
  • saves a json file containing the MVC values to saveDir

See also

  • Uses Run to count runs
  • Uses SerialTrigger to send markers
  • Uses Bar for visual feedback

Tip

The json file is created for the specific target channel and saved within saveDir. Use it for EMG-feedback with feedback

BVR Markers

  • Ready starts at ‘S 6’
  • Go starts at ‘S 14’
  • Feedback ends at ‘S 10’
