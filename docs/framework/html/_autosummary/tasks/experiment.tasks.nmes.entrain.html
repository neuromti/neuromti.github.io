experiment.tasks.nmes.entrain¶

Entrain using peripheral stimulation driven by the neuroConnDC

Documentation¶

experiment.tasks.nmes.entrain.calc_duration(saveDir, stimTarget)[source]¶
experiment.tasks.nmes.entrain.start(nmesTarget='EDC_L', tcsTarget='C4', tacsDuration=3, tacsFreq=20, tacsPhase='undefined', mcsIntensity='single', numberOfTrials=1, numProbes={'pre': 20, 'inter': 20, 'post': 0}, saveDir='C:\\projects\\recording')[source]¶

Notes

Peripheral nerve stimulation with the multiChannelSystems paired to specific phases of transcranial alternating current applied with the neuroConnDC. At the beginning, between trials and at the end of the run, probing burst of NMES are applied.

Parameters:
  • nmesTarget (the target muscle used for) –
  • tcsTarget (the target for tACS) –
  • tacsDuration (the duration of tACS in seconds) –
  • tacsFreq (the frequency of tACS) –
  • tacsPhase (the phase of tACS at which NMES is applied) –
  • mcsIntensity (the stimulation intensity of NMES) –
  • numProbes (a dictionary with pre, inter and post keys linked to integer values, that define the number of test pulse beforepre and post the run as well as between trials) –
  • numberOfTrials (A trial consists of tACS and paired NMES with test pulses applied inbetween.) –
  • saveDir (where the runFile will be stored) –
  • rwskp (a json file containing the EEG workspace) –

side effect

saves a Brainvisionrecorder eeg file to saveDir

See also

  • Uses Run to count runs
  • Uses Serialtrigger to send markers

Tip

Be aware that the stimulators have to be programmed to the desired settings. There is no programmatic control over these devices besides triggering! The multiChannelSystems needs to to be in single mode. Consider especially, that the neuroConn DC needs to listen to its TrigIn and deliver a TrigOut at the desired period and phase. Double-check the hardware setup!

BVR Markers

  • tACS starts at ‘S254’ and/or ‘R 1’
  • A NMES pulse is applied at ‘S 1’
