experiment.tasks.other.rest¶

Perform a resting state measurements

experiment.tasks.other.rest.start(numberOfTrials=1, durationEC=55, durationEO=5, qualifier='', saveDir='C:\\projects\\recording')[source]¶

Performs a resting state measurement

Parameters:
  • numberOfTrials (int) – A trial consists of on eyes open, eyes closed each
  • duratioEC (float) – Duration of EC in seconds
  • durationEO (float) – Duration of EO in seconds
  • qualifier (str) – will be appended to name of the runFile
  • saveDir (str) – where the runFile will be stored

side effect

saves a Brainvisionrecorder eeg file to saveDir

See also

  • Uses experiment.protocol.Run.filename() to count runs
  • Uses experiment.device.Serialtrigger to send markers

Tip

If you only want to record EO, set durationEC to 0 and durationEO to you desired duration (or vice versa)

BVR Markers

  • EO starts at ‘S 6’
  • EC starts at ‘S 10’
  • Run ends witk ‘S 14’
