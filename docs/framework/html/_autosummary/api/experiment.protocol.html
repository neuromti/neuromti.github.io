class experiment.protocol.Cue(Sound, Frame, Duration, trigger=<function register_triggerdevice.<locals>.register_markervalue.<locals>.send_marker>)[source]¶

Class representing a Cue

Parameters:
  • Sound (Sound) – The acoustic part of the cue
  • Frame (Frame) – The visual part of the cue
  • Duration (float) – Duration in seconds
  • Marker (function) – a function sending a trigger on evaluation

Example:

speaker     = device.Speaker()
sound       = stimulus.Sound(speaker, pathToWavFile)
screen      = device.Screen()
frame       = stimulus.Cross(screen)
duration    = 5
interface   = device.SerialTrigger()
marker      = protocol.register_triggerdevice(interface)
cue         = protocol.Cue(sound, frame, duration, marker(6))
interface.connect()
cue.present()
interface.close()

Tip

use the dictionary of cues, sounds and visuals in api

experiment.protocol.register_triggerdevice(td=None)[source]¶

registers a TriggerDevice

Returns:function
Return type:a function for this device

Notes

the returned function in turn registers marker values and returns a function that sends this marker

Example:

m = register_triggerdevice(None)(None)
Marker = register_triggerdevice(HardWareTrigger)
m   = Marker(5) # creates a function to send 5 with this device
m.get() # returns the value of the device

Tip

m = register_triggerdevice(None)(None) sends to a nulldevice
