experiment.stimulus¶

This module contains classes resembling acoustic and visual stimuli

It contains acoustic stimuli, represented by Sound and visual stimuli represented by the general Frame

Visual stimuli¶

Are grouped into whether they are a

  • static Frame, e.g. Mural, Cross or Rectangle
  • Dynamic e.g. Ball or Bar

class experiment.stimulus.Cross(Screen, ZoomFactor=1)[source]¶

class experiment.stimulus.Dynamic(Screen, Target=100, State=0, Tolerance=10)[source]¶

Base class resembling dynamic visual stimuli

Parameters:
  • Screen (Screen) – A display Screen is a necessary requirement
  • Target (float) – value between 0 and 100 defining the target level
  • State (float) – value between 0 and 100 defining the current level
  • Tolerance (float) – value between 0 and 100 defining the range of values accepted around target as sufficient

Caution

as this is the base class, most methods are abstract and need to be implemented, e.g. as in Bar or Ball

Tip

  • you can use print(Dynamic) for debugging purposes
  • will use the default colors of its Screen change these accordingly

class experiment.stimulus.Mural(Screen, Mural='TxT', ZoomFactor=1)[source]¶

Represents white text in the center of a screen

Example

mural = stimulus.Mural(Screen, Mural='Example', ZoomFactor=1)
mural.showNow()
Parameters:
  • Screen (Screen) – A display Screen is a necessary requirement
  • Mural (str) – The string to be shown
  • Zoomfactor (float) – The relative size of the cross, higher values indicate a larger cross

Tip

The font used will be the default font of the Screen
